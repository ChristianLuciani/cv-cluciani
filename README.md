# Christian Luciani — Museo de una Trayectoria

CV interactivo diseñado como recorrido museográfico, con ilusiones ópticas animadas y scroll cinematográfico.

**[→ Ver en vivo](https://christianluciani.github.io/cv)**

## Estructura

```
cv/
├── index.html      ← El CV completo (single file, sin dependencias)
└── README.md
```

## Stack

HTML · CSS · Canvas API · Vanilla JS  
Sin frameworks. Sin dependencias externas. Un solo archivo.

## Ilusiones ópticas incluidas

| Sala | Ilusión | Técnica |
|------|---------|---------|
| 01 · Perfil | Cubo de Necker | Canvas animado — oscila entre dos percepciones |
| 02 · Drahma | Escalera de Penrose | Canvas — movimiento circular infinito |
| 03 · Experiencia | Patrón de interferencia | Canvas — doble fuente de onda |
| 04 · Física | Serpientes Rotantes (Kitaoka) | Canvas estático — ilusión perceptual |
| 04 · Física | Triángulo de Penrose | Canvas animado — pulso imposible |
| 05 · Competencias | Triángulo de Sierpinski | Canvas fractal |
